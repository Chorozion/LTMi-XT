The Cassandra T1 model is a 28-layer transformer with a hidden size of 2048. It uses grouped-query attention with 16 query heads and 4 key-value heads. Position encoding is RoPE. Normalization is RMSNorm. The feed-forward network uses SwiGLU with intermediate size 5632. The vocabulary is 32768 tokens. The mask token id is 32766 and the pad token id is 32767. Default inference uses 12 denoising steps. Training reached a final loss of 2.2561 at epoch 5. Earlier epochs recorded losses of 3.78 at epoch 1, 2.89 at epoch 2, 2.67 at epoch 3, and 2.42 at epoch 4. The released epoch-5 FP16 checkpoint is 2659500664 bytes. The released v2 scratch checkpoint at step 82002 is 16665974950 bytes. Both are split into Git LFS parts because GitHub rejects single LFS objects above 2 GB. The tokenizer file is 2253607 bytes. All artifacts are content-addressed using BLAKE2b. Inference scripts include run_cassandra.py for local CUDA, chunk_gen.py for chunked generation, and serve_ep5.py for a Flask chat-completion endpoint. The repository is licensed Apache 2.0.
